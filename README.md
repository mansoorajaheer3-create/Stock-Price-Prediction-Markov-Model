# Stock Price Prediction using Markov Model

## Overview
This project predicts short-term stock price movements using a Markov Model. Stock prices are classified into three states:

- Increase (I)
- Decrease (D)
- No Change (R)

The model calculates transition probabilities between states and predicts future stock behavior.

## Objectives
- Develop a Markov Model for stock prediction
- Analyze stock movement patterns
- Construct a transition probability matrix
- Predict short-term trends

## Methodology
1. Data Collection
2. Data Preprocessing
3. State Classification
4. Transition Matrix Construction
5. Probability Distribution Analysis
6. Prediction and Evaluation

## Results
- Decrease State Probability: 0.5547
- No Change State Probability: 0.0313
- Increase State Probability: 0.4141
